## 🚀 Quick Start

### Prerequisites

- Node.js (18+)
- npm/pnpm

### Installation

```bash
git clone [repo-url]
cd portfolio
npm install
npm run dev
```

### 🔥 Production Build

```bash
npm run build
npm run preview
```
